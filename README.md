## Installation (Manual - Until Chrome Web Store Release)

Since the extension is not yet available on the Chrome Web Store (coming December 2025), you can install it manually:

### Step-by-Step Installation

1. **Download the Extension**
   - Clone or download this repository to your computer
   - You can download it as ZIP from the GitHub page and extract it

2. **Open Browser Extensions Page**
   - Open Brave/Chrome and go to:
     - `brave://extensions/` (for Brave)
     - `chrome://extensions/` (for Chrome)

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner

4. **Load the Extension**
   - Click the "Load unpacked" button
   - Navigate to the folder where you extracted the extension
   - Select the folder and click "Open"

5. **Start Using**
   - The extension icon should appear in your browser toolbar
   - Click the icon to configure your filter settings
   - Visit Twitter/X and enjoy your filtered experience!

## How to Use

1. **Configure Settings**
   - Click the extension icon in your toolbar
   - Toggle the extension on/off
   - Choose between Dim or Remove mode

2. **Set Up Filters**
   - **Countries Tab**: Select countries you want to filter
   - **Users Tab**: Add specific usernames to block
   - **Keywords Tab**: Add keywords to filter from tweet content

3. **Monitor Activity**
   - Use the statistics panel to see filtering activity
   - Click "Rescan Page" to refresh filtering
   - Reset stats when needed
